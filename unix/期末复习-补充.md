#

## Unix

### 用户ID和组ID

| 类型                   | 描述                                                         | 缩写/术语                 |
|------------------------|--------------------------------------------------------------|-------------------------|
| 1. 用户和组标识ID       | Linux系统中每个用户和组的唯一标识ID。                         | UID/GID                 |
| 2. 文件所有者相关ID      | 拥有某文件的用户的ID和该用户所属组的ID。                     | 文件所有者ID/文件所有者组ID |
| 3. 实际用户ID和实际组ID  | 运行该进程的用户的ID和该用户所属的组ID。                     | 实际用户ID/实际组ID       |
| 4. 有效用户ID和有效组ID  | 用于文件访问权限检查的进程ID。大多数情况下等于实际用户/组ID。 | 有效用户ID/有效组ID       |
| 5. 设置用户ID位和设置组ID位 | 在可执行文件的权限标记中，设置用户ID位表示执行该文件时，进程的有效用户ID变为文件的所有者ID。设置组ID位同理。 | 设置用户ID位/设置组ID位 |
| 6. 保存的设置用户ID和保存的设置组ID | 在执行一个程序时（exec），包含了有效用户ID和有效组ID的副本。 | 保存的设置用户ID/保存的设置组ID |
